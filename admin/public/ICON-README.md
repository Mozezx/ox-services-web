# Ícone PWA (OX Admin)

Para o ícone do app **ocupar todo o círculo** sem bordas brancas:

1. Use uma imagem **512×512 px**.
2. **Fundo** da cor `#0B242A` (ou da marca), **sem** áreas brancas.
3. A **logo** deve preencher o quadro (full-bleed) ou pelo menos o centro ~80% (zona segura maskable).
4. Substitua `logo.png` por esse arquivo e faça novo build do admin.

O `background_color` do manifest já está `#0B242A`; em ícones com transparência, o sistema usa essa cor em vez de branco.
